# dot_files

Dotfiles for a linux setup

## Setup dotfiles

```
git clone https://github.com/busykoala/dot_files ~/dot_files
cd ~/dot_files
chmod +x setup.sh
./setup.sh
```

## Setup manjaro

Get Manjaro with i3wm from `https://manjaro.org/download/i3/`.




```
cd ~/dot_files
chmod +x setup_manjaro.sh
./setup_manjaro.sh
```
